# 📔 LarpNote (MVP Phase)

> **LarpNote** — это современный, быстрый и абсолютно приватный локальный десктопный дневник разработчика. 
> Проект разрабатывается независимой командой как R&D-проект.

---

## 🎯 Что это и зачем?
Ты открываешь приложение, видишь свои текущие ИТ-проекты, добавляешь записи в формате "что сделал сегодня", трекаешь потраченное время и смотришь наглядную статистику продуктивности. 

**Главная фишка:** Всё хранится на 100% локально на ПК пользователя в легковесной базе данных SQLite. Никакого интернета, никаких облачных аккаунтов, личных данных и архивов.

---

## 🛠 Технологический стек (Июнь 2026)
* **Оболочка (Десктоп):** Tauri v2 (Rust-ядро для безопасности и легковесного рантайма).
* **Фронтенд:** React + Vite + Tailwind CSS.
* **Бэкенд:** Python 3 + Flask + Flask-CORS + SQLAlchemy.
* **База данных:** SQLite (хранится локально в виде одного файла).
* **Сборка и CI/CD:** GitHub Actions + PyInstaller (упаковка Python-скриптов в бинарный sidecar внутри Tauri).

### 🛑 Чего НЕТ в MVP (Out of Scope):
Мы не тратим время на: облачную синхронизацию, авторизацию, кастомные темы оформления, экспорт графов, мобильные версии. Это задачи следующих фаз. Только локальный core-функционал.

---

## 🖥 Архитектура MVP (3 Экрана)
1.  **Проекты** — Список твоих ИТ-проектов (Название, цвет, описание).
2.  **Записи** — Таймлайн внутри проекта: дата, сколько времени потратил, теги и поле "что сделано" с поддержкой Markdown-разметки.
3.  **Статистика** — Дашборд: сколько часов за неделю/месяц ушло на каждый проект (простой красивый график).
